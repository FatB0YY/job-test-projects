# job-test-projects

Решениями тестовых заданий от разных IT-компаний.

1. Movies List + back (Цифровые Решения)
2. Todo List (Mindbox)
3. Async List (Nebo)
4. Modal Queue

## Запуск

1. перейти в client, запустить `npm i`, далее `npm run dev`
2. перейти в server, запустить `npm i`, далее `npm run dev`
